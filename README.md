# Hello Reason React

Simple hello world made with Reason React.
It features:

* Reason
* React
* Sass
* Browserify
* MiniB


## Setup

Install dependencies:

    npm i -D

Run dev server:

    npm start

And check the app running on `localhost:8080`.

Or run the production build:

    npm run build

And check the resulting app on the `dist` directory.
